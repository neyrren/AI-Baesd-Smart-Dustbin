AI Classification

Powered by MobileNetV2, a lightweight convolutional neural network fine-tuned on 22,564 waste images for high accuracy sorting.

IoT Integration

ESP32-CAM captures waste images and transmits them wirelessly to the classification server over a local WiFi network.

Live Monitoring

A real-time web dashboard tracks every classification, displaying results, confidence scores, and historical data.

Persistent Storage

Every classification is saved to a local database, ensuring no data is lost between sessions and enabling long-term analysis.

2
